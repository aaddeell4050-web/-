# عادل لتسديد القروض

حلول تمويلية وتسديد قروض لجميع البنوك مع عادل لتسديد القروض، فك إيقاف الخدمات، وتحديث تقارير سمة في جميع أنحاء المملكة.

## المميزات
* تسديد قروض لجميع البنوك
* فك إيقاف الخدمات
* تحديث تقارير سمة
* حلول تمويلية متكاملة

## التشغيل محلياً
لتشغيل المشروع على جهازك:
1. `npm install`
2. `npm run dev`
